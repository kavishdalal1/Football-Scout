# Football Scout AI

A football analytics project built using Python, Pandas, Matplotlib and Seaborn.

## Features

- Position-based player segmentation
- Custom player rating system
- Hidden gem analysis
- Best finisher detection
- Team analysis
- League analysis
- Correlation heatmaps
- Age vs performance analysis

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Outputs

- Top 20 players by position
- Hidden young talents
- Best finisher of the season
- Best attacking and defensive teams
- Best attacking and defensive leagues
- Correlation analysis of football metrics
